# 🎂 Fariha Khan's Birthday Surprise 🎈

A premium, interactive, and beautifully animated birthday celebration website built with Next.js and Framer Motion. This project was designed to create a magical and memorable experience for a special person.

## ✨ Features

- **🕒 Countdown Loader**: A stylish countdown to build anticipation.
- **🎂 Interactive Cake**: A beautiful 3D-styled cake where you can "light" the candle to trigger the celebration.
- **📸 Polaroid Memories**: A premium Polaroid-style photo stack with a "smart-fit" design that ensures no faces are cut off.
- **💌 Special Message**: A personalized, glassmorphism-styled message with floating decorative emojis.
- **🎵 Music & Visuals**: Background music with a pulsing visualizer and interactive confetti effects.
- **📱 Fully Responsive**: Perfectly optimized for mobile, tablet, and desktop screens.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Effects**: [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti)
- **Slider**: [Swiper.js](https://swiperjs.com/)

## 🛠️ Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone [https://github.com/Diya-14/fariha-birthday.git]
   cd fariha-birthday
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📸 Customization

- **Photos**: Replace the images in `public/images/` with your own (`1.jpg`, `2.jpg`, etc.).
- **Music**: Replace `public/birthday.mp3` with your preferred track.
- **Message**: Edit `src/components/screens/MessageScreen.jsx` to change the final personalized message.
